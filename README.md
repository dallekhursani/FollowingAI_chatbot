##  FollowingAI Chatbot API

A professional AI chatbot backend built using **FastAPI** and **Large Language Models (LLMs)**.  
This project demonstrates a clean, production-ready API architecture for multi-turn conversational AI.

---

##  Features

- Multi-turn conversation with memory
- LLM-powered responses (OpenAI GPT or Llama)
- FastAPI backend for easy integration
- Fully modular and extendable
- Ready for deployment on Render, Railway, HuggingFace, or cloud platforms
